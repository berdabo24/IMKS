The working project can be accessed through the Render deployment:

https://imks-y67y.onrender.com

For manual local setup:

1. Install Python - https://www.python.org/ (Ensure that you have added Python to your Windows environmental variables)
2. Open project file (IMKS) as CMD
3. Activate the virtual environment using this command: venv\Scripts\activate
4. Ensure that (venv) is visible at the start of the prompt
5. Install the libraries: pip install -r requirements.txt
6. After it has finished downloading the necessary libraries, run the server using this command: python app.py
7. Open a browser and go to http://127.0.0.1:5000/
